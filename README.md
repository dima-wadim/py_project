Как вы думаете, какие еще типовые вопросы могут быть? Перечислите их..

b.	Вы могли бы мне перезвонить? 
  c   Почему вы выбрали меня?
  d  Зачем вы мне звоните?

  F  Куда можно на вас пожаловаться?
